## About Video-example-react-laravel-pusher

its simple example to get video call and peer to peer calls  using pusher js and react js 

config the database throw env file 
write this commands 
- composer install 
- add connection database to env file 
- npm install & npm run dev  
- php artisan make:auth 
- php artisan serve 

<b> check localhost:8000 </b>
